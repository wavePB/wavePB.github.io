WavePB: Speaker Selection via Inaudible WavePiggyback

Wave PiggyBack is a speaker selection system prototype that records the sounds selectively for multiple smartphone microphones.

Features

  • How does WavePB work?

    WavePB uses deep learning model to seperate speaker's voice and generate the shadow wave indicating the interest speaker. e.g. Focus the specific speaker or hide the sensitive speaker. Then transmit the shadow wave to superpose this acoustic wave on every deployed microphones.
